예약추가
 이름, 연락처, 예약내용 채워서 예약추가하기
 + 고객 메모 채워서 추가(고객 관리의 고객메모에 업데이트 되면 안된다.)
 이미 있는 고객의 경우 고객 이름에 이름이 입력이 되었을 때 고객 메모와 연락처가 자동으로 로딩되는지 확인
 
 예약내용 없이 추가하기
 고객 이름 없이 추가하기
 예약 시작 시간이 끝 시간보다 미래로 세팅한 뒤 추가하기 -> 에러나야 함
 하나 이상의 예약 내용으로 저장하기
 메뉴관리에 추가 한 메뉴와 없는 메뉴로 예약추가하기
 여러개 메뉴 적어놓고 삭제 한 뒤 예약추가하기

 추가 한 담당자로 지정한 뒤 저장하기

 알림톡 사용 후 연락처 입력한 상태로 예약추가 했을 때 알림톡 전송되는지 확인
 알림톡 사용하지 않은 상태에서 연락처 입력한 상태로 예약추가 했을 때 알림톡 전송 안되는지 확인
 연락처 없이 예약추가 할 때 알림톡 전송시도 안하는지 확인

 !!!!!!!!!!고객관리 화면 한번 접속 후 예약추가에서 예약추가하면서 고객추가한 뒤 다시 고객관리 화면 가면 업데이트안되어있는지 확인

예약수정
 드래그로 시간 늘리기
 드래그로 시간 줄이기
 드래그로 시간 이동하기
 
 이름 수정하기 -> 고객관리에서 고객 이름 수정되는지 확인
 연락처 수정하기 -> 고객추가되었는지 확인
 시간 수정하기
 예약내용 수정하기
 담당자 수정하기
 예약상태 수정하기
 예약 상태 노쇼로 수정하고 노쇼 횟수 추가되는지 확인
 예약상태 노쇼에서 정상이나 취소로 수정하고 노쇼 횟수 감소되는지 확인

 !!!!!연락처 있는 고객으로 예약 추가 한 뒤 연락처 없는 애로 바꿨을 때 연락처 없는 애한테 연락처가 생기는지 확인
 -> 고객추가되는지 확인

알림톡 재전송 기능 확인하기

일정 추가
 일정 내용, 예약시간 넣어서 추가하기
 일정내용 없이 추가하기 -> 에러나야 함
 이틀 이상 지속되는 일정 추가하기


노쇼추가

  고객 이름 또는 예약내용으로 검색되는지 확인!!!!!!!!!!!!!!!!
  체크해서 추가 되는지 확인
